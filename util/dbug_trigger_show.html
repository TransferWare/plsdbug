NAME

dbug_trigger_show.pl - Show the state of data using DBUG output of DBUG_TRIGGER triggers.

SYNOPSIS

perl dbug_trigger_show.pl [ -h ] file(s)

DESCRIPTION

This script show the database state after a DBUG_TRIGGER trigger has been fired. All columns will be displayed with their actual data. Data columns will be sorted alphabetically. Since dbug_trigger only shows changes, missing data is retrieved from previous triggers fired.

OPTIONS

h

Display help.

NOTES

EXAMPLES
